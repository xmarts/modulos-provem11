HR Recruitment/Employee - Extended

This module add some information gathering tabs on applicant and employee form where HR manager can enter details.
HR Applicant and Employee with Education/Medical/Employment/Family Details.
This module allow user to pass Employee details, Education details, Family details and Medical checklists from applicant to employee when creating employee from applicant.

Menus List:

  • Recruitment/Configuration/Institutes
  • Recruitment/Configuration/Medical Tests
  • Recruitment/Configuration/Organizations
  • Recruitment/Configuration/Relations

Odoo Job Applicant Form - Past Employment Details

Odoo Job Applicant Form - Family Details

Odoo Job Applicant Form - Medical Checklists

Odoo Applicant - Create Employee Button Extended

When user click on Create Employee button employee is created by Odoo and it will pass the entered data in Education/Family/Medical/Employment from applicant to employee.
